# CLAUDE.md - 易学三合项目上下文

## 项目概述

基于 Cloudflare Pages + Workers 的在线易学应用，包含三大功能模块：
1. **六爻纳甲排盘**：铜钱摇卦 → 纳甲装卦（六亲/世应/六神/伏神/空亡）→ 经典文献对照 → AI深度解读
2. **四柱八字排盘**：输入出生日期时辰 → 排四柱 → 十神/藏干/五行/大运/流年 → AI深度解读
3. **黄历查询**：选日期 → 干支/建除十二神/二十八宿/宜忌/吉神方位/冲煞/彭祖百忌 → AI深度解读

## 核心算法详解

### 1. 日历干支计算

**日柱**：基于儒略日数（JDN）推算
```javascript
function jdn(y, m, d) // 公历→儒略日
function riGZ(y, m, d) // 日柱天干地支，公式：TG[(JDN-1)%10], DZ[(JDN+1)%12]
```

**年柱**：以立春（约2月4日）为界，立春前算上一年
```javascript
function nianGZ(y, m, d) // (y-4)%10→天干，(y-4)%12→地支
```

**月建**：以节气分月，当前用公历近似日期（精度±1天）
```
寅月≈2/4  卯月≈3/6  辰月≈4/5  巳月≈5/6  午月≈6/6  未月≈7/7
申月≈8/7  酉月≈9/8  戌月≈10/8 亥月≈11/7 子月≈12/7 丑月≈1/6
```
**已知精度问题**：节气日期每年有±1-2天偏差，精确版应使用天文算法或查表。

**月干**：年上起月法（甲己之年丙作首…）
```javascript
function yueGan(nianGan, yueZhi) // base表 + 地支偏移
```

**时柱**：日上起时法（甲己还加甲…）
```javascript
function shiGZ(riGan, hour) // hour为24小时制
```

### 2. 六爻纳甲引擎

**摇卦**：模拟三枚铜钱，每枚正（3）反（2），三枚之和 = 6/7/8/9
- 6 = 老阴（阴爻，动爻，变为阳）
- 7 = 少阳（阳爻，静爻）
- 8 = 少阴（阴爻，静爻）
- 9 = 老阳（阳爻，动爻，变为阴）

**八宫归属**：64卦分属八宫（乾坤坎离震巽艮兑），通过八宫生成规则反向查表
```
第1卦：八纯卦（世在六爻）
第2卦：初爻变（世在初爻）
第3卦：初二爻变（世在二爻）
第4卦：初二三爻变（世在三爻）
第5卦：初二三四爻变（世在四爻）
第6卦：初二三四五爻变（世在五爻）
第7卦：游魂（四爻变回，世在四爻）
第8卦：归魂（下卦全变回，世在三爻）
```
世应关系：世爻与应爻间隔3位（shi, ying = (shi+3)%6）

**纳甲规则**（每个经卦的地支分配）：
```javascript
var NJD = {
  乾: {inner:["子","寅","辰"], outer:["午","申","戌"]},  // 甲/壬
  坤: {inner:["未","巳","卯"], outer:["丑","亥","酉"]},  // 乙/癸
  坎: {inner:["寅","辰","午"], outer:["申","戌","子"]},  // 戊/戊
  离: {inner:["卯","丑","亥"], outer:["酉","未","巳"]},  // 己/己
  震: {inner:["子","寅","辰"], outer:["午","申","戌"]},  // 庚/庚
  巽: {inner:["丑","亥","酉"], outer:["未","巳","卯"]},  // 辛/辛
  艮: {inner:["辰","午","申"], outer:["戌","子","寅"]},  // 丙/丙
  兑: {inner:["巳","卯","丑"], outer:["亥","酉","未"]}   // 丁/丁
};
// inner=内卦(初二三爻)天干用yang，outer=外卦(四五上爻)天干用yin
```

**六亲**：以卦宫五行为"我"
- 同我 = 兄弟
- 我生 = 子孙
- 生我 = 父母
- 我克 = 妻财
- 克我 = 官鬼

**六神**：按日干排列，从初爻到上爻
```
甲乙日起青龙，丙丁日起朱雀，戊日起勾陈，己日起螣蛇，庚辛日起白虎，壬癸日起玄武
顺序固定：青龙→朱雀→勾陈→螣蛇→白虎→玄武
```

**空亡**：按日柱所在甲子旬，旬中未用到的两个地支为空亡
```javascript
function getKW(riGan, riZhi) // 返回 [空1, 空2]
```

**伏神**：卦中缺某六亲时，从本宫八纯卦中找该六亲的爻

**动变分析**：
- 化比和：动爻与变爻五行相同
- 化泄：动爻五行生变爻五行
- 化回头生：变爻五行生动爻五行（吉）
- 化克出：动爻五行克变爻五行
- 化回头克：变爻五行克动爻五行（凶）
- 化墓/化绝：变爻地支是动爻五行的墓/绝地

### 3. 八字排盘

**十神**：以日干为"我"，其余七个字与日干的五行关系
```
同类同阴阳=比肩，同类异阴阳=劫财
我生同阴阳=食神，我生异阴阳=伤官
我克同阴阳=偏财，我克异阴阳=正财
克我同阴阳=七杀，克我异阴阳=正官
生我同阴阳=偏印，生我异阴阳=正印
```

**藏干**：每个地支内含1-3个天干
```javascript
var CANG = {子:["癸"], 丑:["己","癸","辛"], 寅:["甲","丙","戊"], ...}
```

**日主旺衰**（简化判断）：同类+生我的五行力量 vs 其余五行力量

**大运排列**：
- 阳年男/阴年女 → 顺排（月柱干支往后推）
- 阴年男/阳年女 → 逆排（月柱干支往前推）
- 每步大运10年

### 4. 黄历系统

**建除十二神**：根据月建地支和日支的偏移确定
```javascript
var jcIdx = ((日支序 - 月支序) % 12 + 12) % 12;
// 0=建 1=除 2=满 3=平 4=定 5=执 6=破 7=危 8=成 9=收 10=开 11=闭
```

**二十八宿值日**：从基准日循环推算
```javascript
// 基准：2000-1-1 = 虚宿
var xiuIdx = ((当前JDN - 基准JDN + 10) % 28 + 28) % 28;
```

**吉神方位**：按日干查表（喜神/财神/福神）

**冲煞**：日支的六冲对应生肖和方位

## AI调用架构

前端支持两种模式（⚙设置面板）：

**模式一：Worker后端**
- 前端 POST → Worker → Worker调用LLM API → 返回结果
- API Key安全存储在Worker环境变量中
- 支持 Anthropic / OpenAI / OpenRouter 三种provider

**模式二：直连API**
- 前端直接调用LLM API（Key存localStorage）
- 支持同上三种provider
- 适合开发调试，不推荐生产使用（Key暴露在浏览器）

**调用流程**：
```
用户点击"🤖 AI深度解读"
→ 读取排盘结果文本（_lastLyText / _lastBzText / _lastHlText）
→ 拼接系统提示词 + 排盘数据 + 用户问题
→ callLLM(systemPrompt, userContent, callback)
→ 结果渲染到对应的 #ly_ai / #bz_ai / #hl_ai 容器
```

## 已知问题与待完善

### 精度问题
1. **节气日期**：当前用固定近似日期，应改用天文算法（如Jean Meeus算法）或查寿星万年历表
2. **真太阳时**：八字排盘未考虑出生地经度校正，精确排盘需要真太阳时转换
3. **农历**：黄历页面的农历月份用地支反推近似，应接入精确的公历↔农历转换算法
4. **大运起运岁数**：当前用简化公式(i*10-6)，精确算法应根据出生日到上/下一个节气的天数÷3

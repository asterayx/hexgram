# 易学经典数据文件说明

## 目录结构

```
data/
├── README.md           # 本文件
├── gaodao.json         # 高岛易断 64卦
├── huangjince.json     # 黄金策分类断语
└── jiaoshi.json        # 焦氏易林 本卦→变卦占辞
```

## 引用方式

- **Android**: `build.gradle.kts` 中 `assets.srcDirs("../data")`，通过 `AssetManager` 加载
- **iOS**: Bundle Resources 引用 `data/` 目录，通过 `Bundle.main.url` 加载
- **Web前端**: 直接 `<script src="classics.js">` 或 fetch JSON

## gaodao.json — 高岛易断

高岛吞象（1832-1914），日本明治时代易学大家，以实占验证周易。

```json
{
  "source": "高岛易断",
  "version": "1.0",
  "hexagrams": {
    "111111": {
      "name": "乾",
      "fullName": "乾为天",
      "judgment": "此卦纯阳刚健，有龙之象...",
      "yao": [
        "初九：潜龙勿用。时位未至...",
        "九二：见龙在田...",
        "九三：终日乾乾...",
        "九四：或跃在渊...",
        "九五：飞龙在天...",
        "上九：亢龙有悔..."
      ]
    }
  }
}
```

**键名规则**: 6位二进制字符串，从初爻到上爻，`1`=阳，`0`=阴。
例如 `"100010"` = 初爻阳、二爻阴、三爻阴、四爻阴、五爻阳、上爻阴 = 屯卦。

**yao 数组**: 固定6个元素，`[0]=初爻, [1]=二爻, ..., [5]=上爻`。

## huangjince.json — 黄金策

传刘伯温作，《卜筮正宗》逐句注解，六爻占卜分类断语总诀。

```json
{
  "source": "黄金策",
  "version": "1.0",
  "categories": [
    {
      "key": "_总论",
      "label": "综合",
      "text": "动静阴阳，反复迁变...",
      "keywords": []
    },
    {
      "key": "求财",
      "label": "求财",
      "text": "财为养命之源...",
      "keywords": ["财", "钱", "利润", "收入", "投资"]
    }
  ]
}
```

**字段说明**:
- `key`: 程序内部键名
- `label`: 显示给用户的名称（用于下拉框）
- `text`: 断语全文
- `keywords`: 用于自动匹配用户问题的关键字（可选）

## jiaoshi.json — 焦氏易林

西汉焦赣著，本卦→变卦占辞，完整版4096条（64×64），当前为精选条目。

```json
{
  "source": "焦氏易林",
  "version": "1.0",
  "entries": {
    "111111": {
      "000000": "大人垂衣裳，逍遥太自乐...",
      "111000": "日出东方，光照万物..."
    }
  }
}
```

**查询方式**: `entries[本卦key][变卦key]` → 占辞文本。
